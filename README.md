# 💼 Human Resource Management System (HRMS)

A full-stack **Human Resource Management System (HRMS)** built using **Node.js, Express, MongoDB**, and a responsive frontend with **HTML, CSS, Bootstrap, and JavaScript**.

This application helps organizations manage employee data, attendance, payroll, and project allocation efficiently.

---

## 🚀 Features

### 🔐 Authentication & Roles
- Role-based access control
- Separate dashboards for:
  - Admin
  - Employee
  - Project Manager
  - Accounts Manager

### 👨‍💼 Employee Management
- Add, update, and delete employee records
- Store education and work experience
- Manage employee profiles

### 🕒 Attendance Management
- Mark attendance
- View attendance history

### 💰 Payroll System
- Salary management
- Payslip generation
- Bonuses and increments

### 📁 Project Management
- Assign and remove employees from projects
- Track employee project involvement

### 📅 Leave Management
- Apply for leave
- Leave tracking and approvals

---

## 🛠️ Tech Stack

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose

**Frontend**
- HTML
- CSS
- Bootstrap
- JavaScript
